# 后台商品管理 - 设计提示词

## 页面概述

总部运营人员管理标准商品库，包括商品的增删改查和适配关系维护。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 新增商品 | 添加新商品 | 跳转表单页 |
| 商品列表 | 表格展示所有商品 | 分页+筛选 |
| 搜索筛选 | 按名称/分类/状态筛选 | 实时筛选 |
| 编辑商品 | 修改商品信息 | 跳转编辑页 |
| 适配车型 | 配置商品适用的车型 | 弹窗选择 |
| 批量操作 | 批量上下架/删除 | 多选+批量操作 |
| 删除商品 | 软删除商品 | 二次确认 |

## 商品信息字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 商品名称 | 文本 | 必填，唯一 |
| 商品图片 | 图片组 | 必填，最多5张 |
| 价格说明 | 文本 | 必填，如"¥2,980起" |
| 项目分类 | 下拉 | 内饰/外观/功能/舒适 |
| 项目介绍 | 富文本 | 必填 |
| 规格说明 | 多行文本 | 施工内容/时长/保修 |
| 适配车型 | 多选 | 关联车型库 |
| 商品状态 | 下拉 | 启用/禁用/草稿 |

## 商品分类

| 分类 | 说明 |
|------|------|
| 内饰 | 内饰改装相关项目 |
| 外观 | 外观贴膜/改装 |
| 功能 | 功能升级改装 |
| 舒适 | 舒适度提升项目 |

## 商品状态

| 状态 | 标签颜色 | 说明 |
|------|----------|------|
| 启用 | 绿色 | 正常展示 |
| 禁用 | 灰色 | 暂时隐藏 |
| 草稿 | 黄色 | 未完成编辑 |

## 设计风格

- **风格**: 表格式管理，表单复杂但清晰
- **色调**: 白色内容区 + 卡片式表单 + 品牌橙色按钮
- **表单**: 标签左对齐，输入框右侧
- **图片上传**: 缩略图预览 + 删除按钮

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 新增按钮 | default/hover | 主按钮 |
| 商品表格 | 默认 | 带复选框 |
| 分类下拉 | default/open | 筛选+表单 |
| 图片上传 | default/uploading/error | 多图上传 |
| 富文本编辑器 | default/focus | 内容编辑 |
| 车型选择器 | 多选网格 | 品牌+车型 |
| 保存按钮 | default/loading/disabled | 表单提交 |
