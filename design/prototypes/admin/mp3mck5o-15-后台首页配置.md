# 后台首页配置 - 设计提示词

## 页面概述

总部运营人员配置小程序首页展示内容，包括Banner、热门推荐、专题等模块。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| Banner配置 | 管理首页轮播图 | 添加/编辑/删除/排序 |
| 热门车型 | 配置首页热门车型入口 | 添加/删除/排序 |
| 热门商品 | 配置首页推荐商品 | 添加/删除/排序 |
| 热门套餐 | 配置首页推荐套餐 | 添加/删除/排序 |
| 专题推荐 | 配置首页专题入口 | 添加/编辑/删除 |
| 预览 | 预览配置效果 | 跳转预览页 |

## Banner 配置

| 字段 | 类型 | 说明 |
|------|------|------|
| Banner图片 | 图片 | 必填，建议750×400 |
| 跳转链接 | 链接 | 可选，商品/专题/外部 |
| 显示排序 | 数字 | 数字越小越靠前 |
| 状态 | 上下线 | 上线/下线 |

## 专题配置

| 字段 | 类型 | 说明 |
|------|------|------|
| 专题名称 | 文本 | 必填 |
| 专题封面 | 图片 | 必填 |
| 专题描述 | 文本 | 可选 |
| 关联商品 | 多选 | 可选关联商品 |
| 状态 | 上下线 | 上线/下线 |

## 设计风格

- **风格**: 模块化配置，所见即所得
- **色调**: 白色内容区 + 模块卡片 + 拖拽排序
- **配置模块**: 折叠卡片，可展开/收起
- **缩略图**: 网格预览，拖拽排序

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 配置模块 | 展开/收起 | 模块化配置 |
| Banner缩略图 | 默认/拖拽 | 网格展示 |
| 图片上传 | default/uploading/error | 单图上传 |
| 商品选择器 | 多选 | 搜索+列表 |
| 车型选择器 | 多选 | 搜索+网格 |
| 状态切换 | 上线/下线 | 开关样式 |
| 预览按钮 | default/hover | 跳转预览 |
