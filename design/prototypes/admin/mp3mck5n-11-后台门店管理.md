# 后台门店管理 - 设计提示词

## 页面概述

总部运营人员管理加盟门店信息，包括增删改查门店基本信息和可售商品配置。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 新增门店 | 添加新门店 | 弹窗/跳转表单 |
| 门店列表 | 展示所有门店 | 分页展示 |
| 搜索筛选 | 按名称搜索、按状态筛选 | 实时筛选 |
| 编辑门店 | 修改门店信息 | 弹窗编辑 |
| 配置商品 | 为门店配置可售商品 | 跳转配置页 |
| 删除门店 | 软删除门店 | 二次确认 |

## 门店信息字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 门店名称 | 文本 | 必填 |
| 门店地址 | 文本 | 必填 |
| 联系电话 | 文本 | 必填 |
| 营业时间 | 时间范围 | 必填 |
| 门店状态 | 下拉 | 营业中/休息中/停业 |
| 门店图片 | 图片 | 门店实景（可选） |

## 门店状态

| 状态 | 标签颜色 | 说明 |
|------|----------|------|
| 营业中 | 绿色 | 正常接单 |
| 休息中 | 黄色 | 暂时休息 |
| 停业 | 灰色 | 已停业 |

## 设计风格

- **风格**: 表格化管理，清晰高效
- **色调**: 白色内容区 + 表格斑马纹 + 操作按钮
- **表格**: 带边框，斑马纹背景
- **操作按钮**: 文字按钮，编辑蓝色/删除红色

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 新增按钮 | default/hover | 右上角主按钮 |
| 搜索框 | default/focus | 带搜索图标 |
| 状态下拉 | default/open | 筛选 |
| 门店卡片 | 营业/休息/停业 | 不同颜色标识 |
| 编辑按钮 | default/hover | 蓝色文字 |
| 删除按钮 | default/hover | 红色文字，需确认 |
| 分页器 | default/hover | 页码+上下页 |
