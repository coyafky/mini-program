# 后台数据统计 - 设计提示词

## 页面概述

总部运营人员查看小程序运营数据统计，包括访问量、预约量、到店量等核心指标。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 时间筛选 | 选择统计时间范围 | 近7天/近30天/自定义 |
| 核心指标 | 展示关键数据卡片 | 数字+趋势 |
| 转化漏斗 | 展示预约转化路径 | 可视化漏斗 |
| 趋势图表 | 展示每日数据变化 | 柱状图+折线图 |
| 导出报表 | 导出统计数据 | Excel导出 |

## 核心指标

| 指标 | 说明 | 计算方式 |
|------|------|----------|
| 访问量 | 小程序PV | 首页PV总和 |
| 预约单量 | 提交预约单数 | 预约表count |
| 到店量 | 确认到店数 | 状态=已确认到店 |
| 完成量 | 已完成施工数 | 状态=已完成 |

## 转化漏斗

| 阶段 | 指标 | 转化率 |
|------|------|--------|
| 访问用户 | 访问小程序的用户数 | 100% |
| 预约提交 | 提交预约单的用户数 | 访问→预约转化率 |
| 到店确认 | 确认到店时间的预约 | 预约→到店转化率 |
| 完成施工 | 最终完成施工的预约 | 到店→完成转化率 |

## 设计风格

- **风格**: 数据可视化，仪表盘风格
- **色调**: 白色内容区 + 数据卡片 + 图表品牌色
- **图表**: 柱状图、折线图、饼图、漏斗图
- **趋势**: 绿色上涨/红色下跌

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 时间选择器 | default/open | 快捷+自定义 |
| 数据卡片 | 涨/跌/平 | 数字+趋势 |
| 转化漏斗 | 默认 | 可视化漏斗 |
| 柱状图 | 默认 | 每日趋势 |
| 饼图 | 默认 | 门店分布 |
| 排行表格 | 默认 | 带进度条 |
| 导出按钮 | default/loading | 导出Excel |
