- [ ] 统一全局请求封装、错误提示、登录态处理

- [ ] 统一页面埋点事件命名
- [ ] 补齐空态、加载态、接口失败态

### 1.2 页面与流程

- [ ] 首页：Banner、快捷入口、热门商品、热门套餐、门店入口
- [ ] 分类页：一级分类、专题入口、跳转商品列表
- [ ] 商品列表页：搜索、排序、筛选、来源参数兼容
- [ ] 门店页：门店列表、搜索、电话、导航、选择门店、回跳
- [ ] 门店商品页：当前门店可售商品/套餐、分类筛选、加预约意向
- [ ] 商品详情页：图文详情、适配车型、价格说明、门店状态判断
- [ ] 预约单页：展示当前意向项目、增删改、切换门店、去提交

- [ ] 我的页：用户信息、我的预约、联系客服、关于我们
- [ ] 我的预约页：按状态筛选、列表、跳转详情
- [ ] 预约详情页：基础信息、状态时间线、处理备注

### 4.2 一期必须补强
