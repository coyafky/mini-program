> 版本：v1.0 | 日期：2026-05-15 | 状态：已确认

## 一、视觉风格定义

### 1.1 整体气质

| 关键词 | 说明 |
|--------|------|
| 专业 | 不像市井路边店，有品牌感 |
| 清爽 | 留白充足，信息密度适中 |
| 汽车服务感 | 参考高端车后市场 UI |
| 可信赖 | 不做廉价促销风，不过度堆优惠 |

**避免的风格：**
- 浓烈渐变促销色（红/紫/橙大字）
- 密密麻麻的信息流
- 廉价阴影和边框
- 过度使用折扣标签

## 二、小程序首页模块规范

### 2.1 搜索栏

**视觉规格：**
- 高度：44px
- 圆角：`--radius-xl`（22px，胶囊形）
- 背景：`#ffffff`
- 边框：1px solid `--color-border`
- 内边距：左右 16px
- 图标：左侧搜索图标 18px，颜色 `--color-text-hint`
- placeholder：`搜索商品 / 套餐 / 案例`，颜色 `--color-text-hint`

**交互：**
- 点击后跳转到搜索页
- 搜索框内不显示清除按钮（无内容时）
- 页面加载时搜索栏固定在顶部，保留安全区

**CSS 参考：**
```css
.search-bar {
  height: 44px;
  border-radius: var(--radius-xl);
  background: #fff;
  border: 1px solid var(--color-border);
  padding: 0 16px;
  display: flex;
  align-items: center;
  gap: 8px;
  font-size: var(--font-size-base);
  color: var(--color-text-hint);
}
```

### 2.2 Banner 1（主视觉轮播）

**视觉规格：**
- 比例：16:9（宽度自适应）
- 圆角：`--radius-lg`（18px）
- 建议实际尺寸：750×422px（移动端 2x Retina）
- 单张文件大小：≤150KB
- 格式：JPG/WebP
- 指示器：底部居中，距底部 12px
- 指示器样式：当前页为圆角矩形条（4×8px），非当前页为圆点（4px）

**交互：**
- 支持手动滑动轮播
- 自动轮播间隔：4秒
- 点击跳转（由后台配置链接）
- 无限循环

**占位图规则：**
- 未加载完成时显示灰色骨架屏（shimmer 动画）
- 加载失败显示品牌色占位图（带 Logo 居中）

**CSS 参考：**
```css
.banner1 {
  width: 100%;
  aspect-ratio: 16/9;
  border-radius: var(--radius-lg);
  overflow: hidden;
}
.banner1 .indicator {
  position: absolute;
  bottom: 12px;
  left: 50%;
  transform: translateX(-50%);
  display: flex;
  gap: 4px;
}
.banner1 .indicator-bar {
  width: 8px;
  height: 4px;
  border-radius: 2px;
  background: rgba(255,255,255,0.8);
  transition: width 0.3s;
}
.banner1 .indicator-bar.active {
  width: 16px;
  background: #fff;
}
```

### 2.4 商品/套餐卡片

**视觉规格：**
- 布局：2列 Grid
- 图片比例：1:1 正方形
- 图片圆角：`--radius-md`（14px）上方两个角
- 卡片圆角：`--radius-lg`（18px）
- 卡片内边距：0（图片撑满宽度，内容区 12px 内边距）
- 阴影：`--shadow-card`

**卡片信息层级（从上到下）：**

| 层级 | 元素 | 规格 |
|------|------|------|
| 1 | 商品图片 | 1:1，宽度 100%，hover 时 scale(1.02) |
| 2 | 标签（可选） | 角标/推荐标，位置在图片左上角 |
| 3 | 商品名称 | 最多2行，超出省略号，字号 14px/600 |
| 4 | 简短描述（可选） | 最多1行，字号 12px，颜色 `--color-text-hint` |
| 5 | 价格 | 字号 18px/700，颜色 `--color-price-accent` |
| 6 | 门店标签（可选） | 字号 11px，绿色文字，图标 12px |

**状态规范：**

| 状态 | 视觉表现 |
|------|----------|
| 默认态 | 白色卡片 + 轻阴影 |
| 按压态 | scale(0.98) + 阴影减弱，150ms |
| 无货/下架 | 图片置灰叠加，叠加层显示"已售罄"标签 |
| 长名称 | 2行截断 + 省略号，不折行 |

## 三、状态规范

### 3.1 加载态（骨架屏）

**使用场景：** 首页首屏所有模块

**实现规范：**
- 骨架元素颜色：`#e8e8e8`（浅灰）
- shimmer 动画：从左到右渐变闪光，颜色从 `#e8e8e8` → `#f5f5f5` → `#e8e8e8`
- 动画时长：1.5s，infinite
- 不使用 spinner/loading 动画
- 各模块骨架形状须与实际内容轮廓一致

```css
/* Shimmer keyframes */
@keyframes shimmer {
  0% { background-position: -200% 0; }
  100% { background-position: 200% 0; }
}
.skeleton {
  background: linear-gradient(90deg, #e8e8e8 25%, #f5f5f5 50%, #e8e8e8 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
}
```

### 3.2 空态

**规则：** 模块整块隐藏，不显示空白占位符。不在首页留下空洞。

**例外：** 门店入口无数据时可显示一行"附近暂无门店，了解更多优惠请到店咨询"。

### 3.3 错误态

**图片加载失败：**
- 显示灰色图标占位（车型 icon 或商品 icon）
- 不显示破损图片图标
- 不阻塞页面渲染

```css
.image-error {
  background: #f0f0f0;
  display: flex;
  align-items: center;
  justify-content: center;
}
.image-error::after {
  content: '';
  width: 32px;
  height: 32px;
  background: #ccc;
  border-radius: 8px;
}
```

### 3.4 网络错误

- 不显示全页错误提示
- 加载失败的模块显示重试按钮（局部）
- 重试按钮样式：文字链接"重新加载"，颜色 `--color-brand`

## 四、素材规格汇总

### 4.1 图片规格表

| 类型 | 推荐尺寸（px） | 比例 | 文件大小 | 格式 | 说明 |
|------|---------------|------|---------|------|------|
| Banner 1 | 750×422 | 16:9 | ≤150KB | JPG/WebP | 最重要，建议 WebP |
| Banner 2 | 750×200 | 3.75:1 | ≤80KB | JPG/WebP | 渐变背景+文案 |
| 商品图 | 400×400 | 1:1 | ≤80KB | JPG/WebP | 封面图，正方形裁切 |
| 套餐图 | 400×400 | 1:1 | ≤80KB | JPG/WebP | 同商品图规格 |
| 案例封面 | 360×270 | 4:3 | ≤80KB | JPG/WebP | 横向场景图 |
| 门店头像 | 160×160 | 1:1 | ≤30KB | JPG/WebP | 门店列表小图 |
| 快捷图标 | 96×96 | 1:1 | ≤20KB | PNG/WebP | 透明背景单色 icon |

### 4.2 裁切规则

- 商品图：中心点裁切，保证主体在正方形内
- 案例封面：左中或右中点裁切，避免主体被切割
- Banner：主体在安全区内（两侧各留 10% 内容缓冲区）
- 车型 icon：留透明 padding 40px，避免贴边

### 4.3 压缩建议

- JPG 质量：75-80%（Banner 75%，商品图 80%）
- WebP：优先使用，支持渐进式加载
- 批量压缩工具建议：TinyPNG / Squoosh CLI
- 图片 CDN 未上线前，先输出通用尺寸规范，不绑定供应商

### 5.2 模块卡片

**视觉规格：**
- 圆角：`--radius-lg`（18px）
- 背景：`#fafbfe`
- 内边距：16px
- 展开/收起图标：右侧 chevron，颜色 `--color-text-hint`
- 展开动画：高度过渡 0.2s ease

**状态：**
| 状态 | 表现 |
|------|------|
| 展开 | 显示完整配置内容 |
| 收起 | 只显示标题栏，内容区域折叠 |

| 无内容 | 标题栏正常显示，添加按钮 |

### 5.3 Banner 网格

**视觉规格：**
- 每张缩略图：正方形，80×80px（后台列表用）
- 圆角：`--radius-sm`（8px）
- 拖拽手柄：缩略图右下角 grip 点
- 拖拽中态：`opacity: 0.7` + `box-shadow: var(--shadow-drag)`

**交互：**
- 鼠标 hover 缩略图时显示编辑/删除按钮
- 拖拽时出现放置指示线（2px 蓝色虚线）
- 排序后自动保存顺序

### 5.4 图片上传

**视觉规格：**
- 上传框：虚线边框 `#d0d0d0`，圆角 `--radius-md`
- 尺寸：宽度 100%，高度 160px
- 上传中：显示蓝色进度条覆盖
- 上传完成：显示缩略图，右上角显示删除按钮

**交互：**
- 点击上传框触发文件选择
- 支持拖拽上传
- 文件 >2MB：提示"图片过大，请重新选择≤2MB的图片"
- 上传失败：红色边框 + 重试按钮

### 5.5 商品/套餐选择器

**实现方式：底部抽屉（Bottom Sheet）**

**视觉规格：**
- 抽屉高度：屏幕 70%
- 顶部：标题 + 关闭按钮
- 内容区：搜索框 + 可滚动列表
- 底部：已选商品缩略展示 + 确认按钮（固定定位）

**搜索与列表：**
- 搜索框：同首页搜索栏样式，placeholder "搜索商品名称"
- 列表项：左侧 Checkbox + 图片缩略（48×48px）+ 名称 + 价格
- 选中态：Checkbox 打勾，背景变 `--color-brand-bg`

**确认按钮：**
- 固定在底部，高度 48px，圆角 `--radius-md`
- 背景：`--color-accent`
- 文字：白色，"确认选择 (N)"
- 已选 N 件商品时显示数量

### 5.6 车型选择器

**布局：**
- 3列 Grid
- 每个车型块：高度 40px，圆角 `--radius-sm`
- 间距：gap 8px

**状态：**
| 状态 | 表现 |
|------|------|
| 未选中 | 背景白色，边框 1px `--color-border` |
| 选中 | 背景 `#fbf2e2`，边框 2px `#d8a04a` |
| 禁用 | 背景 `#f5f5f5`，文字 `#999`，不可点击 |

### 5.7 状态开关

**视觉规格：**
- 尺寸：宽度 42px，高度 24px
- 圆角：12px（全圆角）
- 轨道色（off）：`#e0e0e0`
- 轨道色（on）：`--color-accent`
- 滑块：白色圆形，14px

**动画：** 切换时轨道色 200ms 过渡

### 5.8 保存与发布反馈

**流程：**
1. 点击「保存发布」→ 按钮文字变为"保存中..."
2. 按钮变为灰色，disabled
3. 保存成功 → 按钮闪绿一次（`--color-accent-light` 背景 200ms）→ Toast 提示"发布成功"
4. 保存失败 → 按钮恢复，Toast 提示"保存失败，请重试"

**Toast 提示：**
- 位置：顶部居中，距顶部 80px
- 样式：圆角 `--radius-md`，背景 `#333`，文字白色
- 自动消失：2秒

## 七、关键开发约束

1. **不改变预约主链路**：视觉调整不涉及业务流程变更
2. **不实现复杂动画**：所有动画控制在 300ms 以内，使用 CSS transition
3. **不过度封装**：避免创建过多微小组件，以可维护性优先
4. **响应式适配**：以 375px 为基准，兼容 320px–428px
5. **安全区处理**：顶部状态栏 + 底部 Home Indicator 均需留足安全距离
