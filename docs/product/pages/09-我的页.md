# 我的页 - 设计提示词

## 页面概述

用户个人中心页面，管理个人信息、查看预约入口等。

## 页面结构

```
┌─────────────────────────────────────┐
│                                     │
│  ┌─────────────────────────────┐   │
│  │  [头像]                     │   │
│  │  微信昵称                    │   │
│  │  [手机号: 138****1234]      │   │
│  │  [绑定手机号]                │   │
│  └─────────────────────────────┘   │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────┐ ┌─────────────┐   │
│  │ 我的预约    │ │ 当前门店    │   │
│  │ [数字]      │ │ [门店名]    │   │
│  └─────────────┘ └─────────────┘   │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────────────────────┐   │
│  │ 联系总部              →     │   │
│  ├─────────────────────────────┤   │
│  │ 品牌介绍              →     │   │
│  ├─────────────────────────────┤   │
│  │ 用户协议              →     │   │
│  ├─────────────────────────────┤   │
│  │ 隐私政策              →     │   │
│  └─────────────────────────────┘   │
│                                     │
└─────────────────────────────────────┘
           底部导航栏
```

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 头像 | 微信头像 | 展示 |
| 昵称 | 微信昵称 | 展示 |
| 手机号 | 绑定手机号 | 已绑定显示/未绑定显示绑定入口 |
| 我的预约 | 跳转预约列表 | 显示数量角标 |
| 当前门店 | 显示当前服务门店 | 点击可切换 |
| 联系总部 | 跳转联系方式 | 外链/客服入口 |
| 品牌介绍 | 跳转品牌介绍页 | 了解品牌 |
| 用户协议 | 跳转协议页 | 条款 |
| 隐私政策 | 跳转政策页 | 条款 |

## 用户状态

| 状态 | 展示内容 |
|------|----------|
| 已登录已绑定 | 头像+昵称+手机号 |
| 已登录未绑定 | 头像+昵称+绑定按钮 |
| 未登录 | 登录按钮 |

## 设计风格

- **风格**: 简洁个人中心
- **色调**: 白色背景 + 深色文字 + 灰色辅助
- **头像**: 圆形，居中
- **菜单项**: 列表式，右箭头指示

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 用户头像 | 默认 | 圆形 |
| 昵称 | 默认 | 文字 |
| 手机号 | 已绑定/未绑定 | 加密/绑定入口 |
| 预约入口 | 有数字/无数字 | 快捷入口 |
| 门店入口 | 已选/未选 | 显示门店名 |
| 菜单项 | default/hover | 带箭头列表 |
| 退出登录 | 红色文字 | 底部可选 |

## 技术要求

- 用户信息自动拉取
- 手机号部分加密显示
- 菜单项点击跳转
- 支持退出登录
