## 使用说明

### 使用流程

1. **阅读页面提示词**：打开对应的 `.md` 文件
2. **提取设计要求**：按照文档中的结构、组件、交互要求
3. **生成原型设计**：使用 Figma/设计工具创建可交互原型
4. **输出设计稿**：提供设计稿源文件或图片

### 设计规范

| 规范项 | 要求 |
|--------|------|
| 设计工具 | Figma（推荐）/ Sketch / Adobe XD |
| 基准尺寸 | 375px × 812px（iPhone X/11/12/13） |
| 颜色模式 | #FF6B00 品牌橙 + #1A1A1A 深色 + #FFFFFF 白色 |
| 字体 | 系统默认无衬线字体 |
| 图标 | Lucide Icons / 微信原生图标 |

## 设计原则

1. **一致性**：所有页面遵循统一的设计语言
2. **简洁性**：突出核心功能，减少干扰
3. **可用性**：交互直观，状态明确
4. **响应式**：适配主流 iPhone 屏幕尺寸

## 输出要求

每个页面需要输出：
- [ ] 完整页面布局设计
- [ ] 所有组件状态定义
- [ ] 点击/触摸交互说明
- [ ] 页面跳转逻辑标注
- [ ] 设计稿源文件或高清图片

## 注意事项

⚠️ **重要提示**：
- 所有价格仅为示例，实际价格由后台配置
- 门店信息由后台配置，界面展示真实数据
- 用户需先选择门店才能浏览商品
- 预约是核心转化目标，按钮需醒目
