# 预约页（我的预约）- 设计提示词

用户查看和管理自己的预约单列表页面。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 状态Tab | 全部/待处理/已联系/已确认/已完成/已关闭 | 切换筛选 |
| 预约单列表 | 展示预约单卡片 | 下拉刷新 |
| 预约单卡片 | 显示关键信息+状态 | 点击进入详情 |
| 查看详情 | 进入预约单详情页 | 按钮 |

## 预约单状态

| 状态 | 标签颜色 | 说明 |
|------|----------|------|
| 待处理 | 橙色 | 总部尚未处理 |
| 已联系 | 蓝色 | 已联系客户或通知门店 |
| 已确认到店 | 绿色 | 确认了到店时间 |
| 已完成 | 灰色 | 已到店施工完成 |
| 已关闭 | 深灰色 | 无效单/取消/未到店 |

## 预约单卡片信息

| 信息 | 显示位置 |
|------|----------|
| 状态标签 | 卡片右上角 |
| 门店名称 | 卡片标题 |
| 预约项目 | 主要内容 |
| 预约时间 | 主要内容 |
| 提交时间 | 次要信息 |

## 设计风格

- **风格**: 列表清晰，状态明显
- **色调**: 白色背景 + 彩色状态标签 + 灰色次要文字
- **Tab栏**: 横向滚动，选中态下划线
- **卡片**: 白色卡片，左侧状态色条

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 状态Tab | default/selected | 横向滚动 |
| 预约卡片 | 待处理/已联系/已确认/已完成/已关闭 | 不同颜色标识 |
| 状态标签 | 颜色区分 | 圆角标签 |
| 查看按钮 | default/hover | 文字按钮 |
| 空状态 | 无数据时 | 友好提示 |
| 下拉刷新 | loading | 刷新动画 |

## 技术要求

- Tab支持横向滚动
- 下拉刷新列表
- 记住用户上次查看的Tab
- 实时更新预约状态
