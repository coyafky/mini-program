# 门店页 - 设计提示词

## 页面概述

门店列表页面，用户必须先选择服务门店才能浏览商品和预约。

## 页面结构

```
┌─────────────────────────────────────┐
│         门店页标题 "选择门店"        │
├─────────────────────────────────────┤
│  当前服务门店 (已选时显示)           │
│  ┌─────────────────────────────┐   │
│  │ 门店名称    [切换]           │   │
│  │ 地址 · 距离 · 营业时间       │   │
│  └─────────────────────────────┘   │
├─────────────────────────────────────┤
│         门店列表                     │
│  ┌─────────────────────────────┐   │
│  │ 🏪 门店名称                  │   │
│  │ 📍 地址                      │   │
│  │ 📞 电话    🕐 营业时间       │   │
│  │              [进店选购] →    │   │
│  └─────────────────────────────┘   │
│                                     │
│  ┌─────────────────────────────┐   │
│  │ 🏪 门店名称                  │   │
│  │ 📍 地址                      │   │
│  │ 📞 电话    🕐 营业时间       │   │
│  │              [进店选购] →    │   │
│  └─────────────────────────────┘   │
└─────────────────────────────────────┘
           底部导航栏
```

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 当前门店 | 显示已选门店信息 | 可快速切换 |
| 门店搜索 | 按名称/地址搜索 | 实时搜索 |
| 门店列表 | 展示所有在营门店 | 卡片式展示 |
| 选择门店 | 点击选择服务门店 | 选中态高亮 |
| 进店选购 | 进入该门店商品页 | 必须先选门店 |

## 门店信息展示

| 信息 | 内容 |
|------|------|
| 门店名称 | 品牌+门店编号/名称 |
| 门店图片 | 门店实景图（可选） |
| 地址 | 详细地址 + 距离（可选） |
| 电话 | 点击可拨打 |
| 营业时间 | 格式：09:00-18:00 |
| 门店状态 | 营业中/休息中 |

## 设计风格

- **风格**: 信任感、官方感
- **色调**: 白色背景 + 深色文字 + 品牌橙色按钮
- **门店卡片**: 白底卡片，清晰分隔信息
- **CTA按钮**: 醒目的"进店选购"按钮

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 当前门店卡 | selected | 显示已选门店 |
| 切换按钮 | default/hover | 更换门店入口 |
| 门店卡片 | default/hover | 完整门店信息 |
| 电话图标 | default/tap | 点击拨打 |
| 进店按钮 | default/hover/disabled | 需先选门店 |
| 门店标签 | 营业中/休息中 | 状态指示 |

## 重要提示

⚠️ **必须先选择门店**：用户选择门店后，系统记住当前服务门店，后续所有商品浏览和预约都以所选门店为准。

## 技术要求

- 门店列表支持下拉刷新
- 门店卡片点击区域合理划分
- 电话拨打使用微信能力
- 记住用户上次选择的门店
