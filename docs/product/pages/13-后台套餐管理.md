## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 新增套餐 | 添加新套餐 | 跳转表单页 |
| 套餐列表 | 展示所有套餐 | 分页展示 |
| 搜索筛选 | 按名称/状态筛选 | 实时筛选 |
| 编辑套餐 | 修改套餐信息 | 跳转编辑页 |
| 配置项目 | 配置套餐包含的商品 | 选择商品 |
| 适用车型 | 配置套餐适用的车型 | 选择车型 |
| 删除套餐 | 软删除套餐 | 二次确认 |

## 套餐信息字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 套餐名称 | 文本 | 必填，唯一 |
| 套餐图片 | 图片 | 必填，最多3张 |
| 套餐价格 | 数字 | 必填，单位元 |
| 套餐简介 | 文本 | 必填，一句话描述 |
| 包含项目 | 多选 | 关联商品库 |
| 适用车型 | 多选 | 关联车型库 |
| 套餐状态 | 下拉 | 启用/禁用 |

## 套餐项目计算逻辑

- 显示所有已选项目的原价总和
- 显示套餐设置的优惠价格
- 显示节省金额

## 设计风格

- **风格**: 与商品管理一致，表单复杂
- **色调**: 白色内容区 + 卡片式表单
- **已选项目**: 卡片列表，可移除
- **价格计算**: 突出显示优惠

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 套餐卡片 | 启用/禁用 | 显示已选项目摘要 |
| 项目选择器 | 多选列表 | 按分类筛选 |
| 已选项目卡片 | 默认 | 显示价格+移除 |
| 车型选择器 | 多选网格 | 与商品管理一致 |
| 价格计算器 | 自动计算 | 原价/优惠价/节省 |

## 输出要求

生成可交互的原型稿，包含：
- [ ] 套餐列表展示
- [ ] 新增/编辑套餐表单
- [ ] 包含项目配置
- [ ] 适用车型选择
- [ ] 价格自动计算
