# 后台预约单管理 - 设计提示词

## 页面概述

总部运营人员查看和处理预约单，更新预约状态和填写处理备注。

## 功能要求

| 模块 | 功能描述 | 交互要求 |
|------|----------|----------|
| 预约单列表 | 展示所有预约单 | 分页+筛选 |
| 状态筛选 | 按状态筛选 | 全部/待处理/已联系/已确认/已完成/已关闭 |
| 门店筛选 | 按门店筛选 | 全部/各门店 |
| 日期筛选 | 按日期范围筛选 | 开始-结束 |
| 搜索 | 按单号/手机号搜索 | 实时搜索 |
| 处理预约 | 更新状态+填写备注 | 弹窗处理 |
| 查看详情 | 查看完整预约信息 | 跳转详情页 |
| 导出 | 导出预约单数据 | Excel导出 |

## 预约单状态

| 状态 | 标签颜色 | 操作 |
|------|----------|------|
| 待处理 | 橙色 | 处理 |
| 已联系 | 蓝色 | 处理 |
| 已确认到店 | 绿色 | 查看 |
| 已完成 | 灰色 | 查看 |
| 已关闭 | 深灰色 | 查看 |

## 设计风格

- **风格**: 表格+详情并重，处理高效
- **色调**: 白色内容区 + 状态色标签 + 橙色操作按钮
- **状态标签**: 彩色圆角标签，醒目区分
- **时间线**: 左侧竖线，处理记录清晰

## 组件清单

| 组件 | 状态 | 说明 |
|------|------|------|
| 状态筛选 | default/open | 彩色标签 |
| 门店筛选 | default/open | 下拉选择 |
| 日期选择器 | default/range | 日期范围 |
| 预约单表格 | 默认 | 带复选框 |
| 状态标签 | 待处理/已联系/已确认/已完成/已关闭 | 颜色区分 |
| 处理弹窗 | 默认 | 表单弹窗 |
| 处理记录 | 默认 | 时间线展示 |
